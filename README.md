## Project Description

This portfolio shows a collection of my projects and skills as a web developer using the MERN stack (MongoDB, Express.js, React, Node.js). The design is responsive, adapting to different devices and screen sizes. It includes a dropdown menu for easier navigation on small screens, styled using Material-UI.

## Technologies Used

- **HTML5**: Basic structure of the site.
- **CSS3**: Styling and layout of the site.
- **JavaScript**: Interactivity and dynamic functionalities.
- **React**: Library for building user interfaces.
- **Material-UI**: Component library for React.
- **Node.js**: Runtime environment for the server.
- **Express.js**: Framework for building web applications in Node.js.
- **MongoDB**: NoSQL database for data storage.
- **Mongoose**: ODM for MongoDB to simplify database interactions.
- **Redux**: State management for React applications.
- **JWT**: Authentication and authorization.

## Installation & Setup

To run this portfolio locally, follow these steps:

1. **Clone the Repository**

   ```bash
   git clone https://github.com/Sebasbh/Portfolio-2.0.git
   ```

2. **Navigate to the Project Directory**

   ```bash
   cd Portfolio-2.0
   ```

3. **Install Backend Dependencies**

   ```bash
   cd backend
   npm install
   ```

4. **Install Frontend Dependencies**

   ```bash
   cd ../frontend
   npm install
   ```

5. **Set up Environment Variables**

   Create a `.env` file in the `backend` directory with the necessary configuration, such as:

   ```env
   MONGO_URI=your-mongodb-uri
   PORT=5000
   ```

6. **Run the Backend**

   ```bash
   cd backend
   npm start
   ```

7. **Run the Frontend**

   ```bash
   cd ../frontend
   npm start
   ```
