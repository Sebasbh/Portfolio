# Frontend - Portfolio

This is the frontend for my personal portfolio, built with React and Material-UI.

## Available Scripts

In the `frontend` directory, you can run:

### `npm start`

Runs the app in development mode at [http://localhost:3000](http://localhost:3000).

### `npm run build`

Builds the app for production to the `build` folder.

## Technologies

- React
- Material-UI
- React Router
- Axios

## Project Description

This frontend displays my web development projects and skills in a responsive, modern interface, featuring adaptive navigation and interactive components.
